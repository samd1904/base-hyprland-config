# base-hyprland-config
A super simple hyprland setup, can be used as is or as a jumping off point to start your own rice

This repo is a collection of my dotfiles for the following packages

hypr - wm & compositor using wayland
kitty - terminal
waybar - status bar

The config are synced using gnu stow.
To make use of these dotfiles you must do the following

    - Install gnu stow on your respective distro
    - Fork and then clone this repo in your system
    - cd into the repo folder
    - run the command `stow --dotfiles <package-name>` (package name can be the name of any packages dotfiles you want to copy) (Make sure that package is installed and working in your system first)

I am actively looking to improve up on this custom config so make sure to check back in from time to time.
